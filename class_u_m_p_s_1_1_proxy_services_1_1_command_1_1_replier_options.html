◆ setPollingTimeOut()

void UMPS::ProxyServices::Command::ReplierOptions::setPollingTimeOut ( const std::chrono::milliseconds &  pollingInterval)

Sets the polling interval for the process that communicates with the module registry.

Parameters
[in]timeOutThe polling timeout. After this time the module registry polling process can check other things, like if the program is shutting down. This must be a balance between being too short which unnecessarily chews cycles and being too long so as to make this utility unresponsive.
Exceptions
std::invalid_argumentif the polling interval is negative.
