Detailed Description

To identify items (picks, events, origins, amplitudes, etc.) in the processing pipeline we must assign them a unique object identifier. The challenge in a distributed setting is that we need a single entity that performs a lock-increment-unlock operation. This is the mechanism for requesting an identifier from that entity.
