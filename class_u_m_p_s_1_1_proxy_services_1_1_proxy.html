Detailed Description

This defines the ROUTER/DEALER proxy. This is an intermediary message layer to which (1) clients send requests to the frontend, then the request is (2) propagated to the backend and the servers.
The servers then (3) send a reply to the backend which is then (3) propagated to the frontend, and (4) finally the reply is received by the original client.
