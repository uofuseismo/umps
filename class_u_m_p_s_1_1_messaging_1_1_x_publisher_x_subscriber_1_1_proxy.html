Detailed Description

A ZeroMQ proxy to be used in the XPUB/XSUB pattern.

The XPUB/XSUB pattern is an extension of the PUB/SUB pattern. The proxy is an intermediary which can forward messages from multiple publishers to multiple subscribers. Conceptually, data flows from the frontend to the backend. Hence, for a pub/sub pattern publishers will connect to the frontend and subscribers will connect to the backend.
